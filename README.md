## ที่มาของปัญหา
เนื่องด้วยสถานการณ์โควิด19 ทำให้คนเรานั้นไม่สามารถออกไปซื้อสินค้าได้และไม่ปลอดภัยจากโรคจึงสร้างแอปพลิเคชั่นขึ้นมาเพื่ออให้ปลออดภัยต่อโรคโควิด19 และยังสะดวกปลอดภัยอีกด้วย
